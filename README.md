# GithubUserSearch

![logo](icon.png)

:octocat:GitHub 유저 목록을 검색하여 보여주는 안드로이드 샘플 앱

## 구현 체크리스트

* [x] GitHub Search API 연동
* [x] API 사용자 목록 RecyclerView UI 구현
* [x] 유저 검색 기능 구현
* [x] 유저 아바타 이미지 출력
* [x] 유저 목록 아이템뷰 UI 구현 (아바타, 아이디, 스코어)
* [x] 유저 클릭시 레포지토리 URL로 외부 브라우저 실행
